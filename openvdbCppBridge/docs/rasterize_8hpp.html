Rasterize a list of particles into a level set and write it to a .vdb file. The files are output in the results directory with the given name and iteration number.

Parameters
positionsthe list of particle positions for one iteration
fileNamethe name of the file to output (without extension) (fluid by default)
iterationthe iteration number (0 by default) to append to the file name
shouldGenerateObjFileswhether or not to generate .obj files (false by default)
